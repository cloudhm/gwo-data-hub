## 汇总

- **数据表总数**：100 张  
- **命名规则**：表名均为 `ods_lingxing_*`，对应 Prisma 模型为 `LingXing*`。  
- **软删除**：多数表含 `archived` 字段，增量/全量同步采用“软删后插入”策略。  
- **关联**：上述表均通过 `accountId` 关联到 `ods_lingxing_accounts`（领星账户）。

如需某张表的字段说明或同步逻辑，可结合 Prisma schema（`prisma/schema.prisma`）与对应服务代码查阅。
